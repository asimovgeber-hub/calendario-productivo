# 📅 Calendario Productivo v3.0

Un calendario inteligente para gestionar tareas, estudio y hábitos diarios con sistema de seguimiento visual.

## 🌟 Demo en Vivo

**👉 [USAR EL CALENDARIO AQUÍ](https://TU-USUARIO.github.io/calendario-productivo/)**

*Reemplaza `TU-USUARIO` con tu nombre de usuario de GitHub*

## ✨ Características

### 📋 Trabajos (Azul)
- Crear tareas con materia, actividad y fecha límite
- 3 niveles de prioridad: Bajo, Medio, Alto
- Contador de días restantes
- Marcar como completada

### 📚 Estudio (Rojo)
- Define hasta 7 materias con jerarquía:
  - ♠ Picas (Alta): 2 materias - estudio diario
  - ♣ Tréboles (Media): 2 materias - día de por medio
  - ♥ Corazones (Baja): 3 materias - una vez por semana
- Distribución automática de horarios de estudio

### 🎯 Hábitos Diarios
6 hábitos a seguir:
- 🛏️ Salud de sueño • 📖 Estudio • 🪶 Anki • 📚 Lectura • 🧴 Cuidado personal • 🎸 Guitarra

**Código de colores** según % completado:
- 🟣 Morado: 100% + ⭐ Estrella
- 🔵 Azul: 85%
- 🔷 Cian: 67%
- 🟢 Verde: 50%
- 🟠 Naranja: 33%
- 🔴 Rojo: 17%
- ⚫ Negro: 0%

### 🆕 Nuevas Funcionalidades v3.0

#### 📌 Recordatorios Visibles
Texto visible directamente en el día con diseño compacto naranja

#### 🔀 Flujogramas Integrados
Mini-preview con los primeros 4 nodos directamente en el día

#### ⚫ Auto-Negro
Marca automáticamente días pasados no registrados como 0%

## 🚀 Uso Inmediato

### Opción 1: Usar en línea (Recomendado)
Simplemente abre: **https://TU-USUARIO.github.io/calendario-productivo/**

### Opción 2: Clonar y usar localmente

#### Windows:
```bash
git clone https://github.com/TU-USUARIO/calendario-productivo.git
cd calendario-productivo
iniciar.bat
```

#### Linux / Mac:
```bash
git clone https://github.com/TU-USUARIO/calendario-productivo.git
cd calendario-productivo
chmod +x iniciar-linux.sh
./iniciar-linux.sh
```
